Step-By-Step Subtraction

When subtracting larger numbers, we follow a clear path:

  1. 1 Line up the numbers by place value
  2. 2 Subtract the ones digits
  3. 3 Subtract the tens digits
  4. 4 Continue with hundreds, thousands, etc.

Clear Path Subtraction Tips

  • üåü Line up carefully - Always line up the digits by place value (ones under ones, tens under tens).
  • üåü Work right to left - Always start with the ones place and move left.
  • üåü Check your work - You can check your answer by adding: if 375 - 124 = 251, then 251 + 124 should equal 375!
  • üåü Watch for borrowing - If you can't subtract (like 2-4), you'll need to borrow from the next digit. We'll learn this on the Borrowing Bridge!
