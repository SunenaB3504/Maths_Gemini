Commutative Property

When you add numbers, you can switch their order and still get the same answer!

a + b = b + a

This means:

  • 3 + 5 = 5 + 3
  • 12 + 7 = 7 + 12
Example:

4 + 9 = 13

4 + 9 = 13

9 + 4 = 13

9 + 4 = 13

The order doesn't matter - the answer is still 13!

Associative Property

When adding three or more numbers, you can group them in any way and still get the same answer!

(a + b) + c = a + (b + c)

This means:

  • (2 + 3) + 4 = 2 + (3 + 4)
  • (5 + 2) + 8 = 5 + (2 + 8)
Example:

(3 + 5) + 2 = 8 + 2 = 10

( 3 + 5 ) + 2 = 10

3 + (5 + 2) = 3 + 7 = 10

3 + ( 5 + 2 ) = 10

The grouping doesn't matter - the answer is still 10!

Identity Property (Zero)

When you add zero to any number, the answer is always that number!

a + 0 = a

This means:

  • 7 + 0 = 7
  • 42 + 0 = 42
  • 0 + 156 = 156
Example:
9 + 0 = 9

Adding zero doesn't change the number!

How Are These Properties Useful?

Mental Math:

To add 8 + 27, you can use the commutative property and think 27 + 8 instead.

Then add: 27 + 8 = 35 (It's often easier to add a small number to a big number)

Making Problems Easier:

To add 14 + 25 + 6, you can use the associative property:

14 + 25 + 6 = 14 + (25 + 6) = 14 + 31 = 45

(It's easier to add 25 + 6 first to get to a multiple of 10)

Finding Patterns:

These properties help us recognize patterns and solve problems faster!
