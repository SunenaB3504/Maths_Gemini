Addition Tips

  • Counting On: Start with the bigger number and count up by the smaller number. For 3 + 2, start at 3 and count: "4, 5".
  • Use Your Fingers: Hold up fingers for each number, then count them all.
  • Make 10: If one number is close to 10, make 10 first, then add the rest. For 9 + 4, think 9 + 1 = 10, then 10 + 3 = 13.
  • Doubles: Learn addition doubles like 2+2, 3+3, etc. They're easy to remember!

Fun Addition Facts!

  • Adding zero to any number gives you the same number. For example, 7 + 0 = 7.
  • Addition is commutative: 4 + 2 gives the same answer as 2 + 4.
  • The plus sign (+) was first used in 1489!
