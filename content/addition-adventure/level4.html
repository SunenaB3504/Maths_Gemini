What is Carrying?

When we add digits and get a sum of 10 or more, we need to carry the extra tens to the next column.

This is also called regrouping. Let's see how it works!

Carrying Tips

  • Always start with the ones place (rightmost digit)
  • Write down the carry above the next column to help remember it
  • Remember to add the carry when working on the next column
  • Check your work by adding the digits in each column again
  • Estimate first! Round the numbers and add to check if your answer makes sense

When Do We Use Carrying in Real Life?

  • Adding money ($45 + $67 = $112)
  • Finding the total weight of items (34 kg + 78 kg = 112 kg)
  • Calculating how many pages you've read (126 pages + 87 pages = 213 pages)
  • Adding distances on a map (156 miles + 95 miles = 251 miles)
