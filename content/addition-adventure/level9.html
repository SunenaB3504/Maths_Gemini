What are Missing Digit Problems?

In these special problems, one digit within a number is hidden. Your job is to figure out what digit belongs in the empty space.

Look for the ? - that's your clue to find the missing digit!

Example 1: Missing Digit in First Number
3? + 25 = 59

To solve: We need to find what digit replaces the ? mark

If ? = 4, then 34 + 25 = 59

So the missing digit is 4

Example 2: Missing Digit in Second Number
46 + 2? = 73

To solve: We need to find what digit makes this addition correct

46 + 27 = 73

So the missing digit is 7

Example 3: Missing Digit in Sum
37 + 45 = 8?

To solve: Find the actual sum, then identify the missing digit

37 + 45 = 82

So the missing digit is 2

How to Solve Missing Digit Problems

You can use these strategies to find the missing digit:

  • Calculate what you know: Solve the parts of the addition that you can.
  • Work backward: If you know the sum and one complete addend, find what the other addend must be.
  • Try different values: Test digits from 0-9 to see which one works.

Tips for Missing Digit Problems

  • Pay attention to place values: Note which place (ones, tens, hundreds) has the missing digit.
  • Check your answer: Always substitute your digit back into the original problem to make sure it works.
  • Consider carries: Remember that digits in one place can affect other places if there's carrying.
  • Think step-by-step: Break down the problem into smaller parts to help find the missing digit.
